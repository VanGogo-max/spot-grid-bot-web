 # 📊 Spot Grid Bot за Android (Termux)

Автоматизиран бот за търговия на **MEXC, Gate.io, KuCoin, CoinEx** — без KYC, с риск 1%, прави сделки на всеки ~20 минути.

---

## ✅ Поддържани борси

- [MEXC](https://mexc.com) — без KYC
- [Gate.io](https://gate.io) — без KYC
- [KuCoin](https://kucoin.com) — без KYC (до известна степен)
- [CoinEx](https://www.coinex.com) — без KYC

---
## 📱 Инсталация за Android (Termux)


1. Инсталирай **Termux** от F-Droid
2. Отвори Termux и изпълни:
   ```bash
   pkg update && pkg install git python -y
git clone https://github.com/VanGogo-max/spot-grid-bot-android.git
cd spot-grid-bot-android
pip install -r requirements.txt
chmod +x run_bot.sh'''
